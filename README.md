#	Interactive 3D Visualization for Energy Planning
Computer Vision  •  Geodata  •  Godot Engine  •  Renewable Energy  •  Digital Twin

This project sits at the intersection of physical hardware and immersive 3D software. The subject is a Godot Engine-based visualization tool that uses geodata to represent complex digital twins of landscapes for planning scenarios — such as renewable energy site selection. The core concept is an interactive 'Gaming Table': a horizontal display surface on which physical tokens represent energy infrastructure elements, tracked by a camera in real time and reflected in the 3D simulation.  The intern/student connects computer science with engineering skills. The system uses a depth camera (e.g., Intel RealSense) and image recognition algorithms to track physical objects on the table and transfer these inputs into the simulation.

## Tasks
*	System Setup and Deployment: Set up the software prototype on high-performance hardware; learn the Godot Engine and plugins for processing large geodata sets (raster and vector data); ensure a stable runtime environment.
*	Hardware Construction and 3D Printing: Design and fabricate (via CAD and 3D printing) a stable mounting bracket for the projector and depth camera, enabling precise physical alignment and projection calibration.
*	Computer Vision and Optimization: Deepen the image recognition logic; stabilize token detection (color/size-based); develop a self-calibration routine to automatically reference the camera system to the projected surface.
